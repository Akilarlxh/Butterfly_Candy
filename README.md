# 糖果屋魔改合集
本项目为糖果屋魔改教程的测试Demo，在本地运行以下指令后即可开始使用。
```bash
git clone https://github.com/Akilarlxh/Butterfly_Candy.git
cd .\Butterfly_Candy\
npm install
```
整合包内提供的`menu.sh`为简易脚本，双击运行。涉及github仓库操作的选项需要您修改脚本中的仓库URL。
